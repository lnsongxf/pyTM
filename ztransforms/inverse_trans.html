The Inverse Z-Transform¶

Given a z-transform \(X(z)\) in the frequency domain, it is sometimes useful to return to the corresponding time sequence \(x(k)\). This is where the inverse z-transform comes in; it moves you from the frequency domain back to the time domain.

inverse z-transform
The inverse z-transform is defined as: \(x(k) = \mathcal{Z}^{-1}[X(z)] = \frac{1}{2 \pi i} \oint_C X(z) z^{k-1} \partial z\) where the countour integration can be evaluated using the Cauchy Residue Theorem. The integration contour should enclose all singularities of \(X(z)\).

Considering the complicated form of the inverse z-transform, one might hope for easier ways to calculate the inverse z-transform than using the definition. Luckily, there exist other methods for finding the inverse z-transform. We will briefly examine a technique where we decompose the function into a combination of functions found on the table and then look up the corresponding inverse z-transform for each part.

Steps:

  1. Break up the function through partial fraction expansion of \(G[z] = \frac{F[z]}{z}\) (This will help us get a constant term that we need).
  2. Multiply by \(z\) to get back to \(F[z]\).
  3. Put each piece into a recognizable form.
  4. Transform each piece.

We will do a quick example of how to solve by inspection using the table above.

First Example¶

Find the inverse z-transform of the function \(X[z] = \frac{8z - 19}{(z-2)(z-3)z}\). We will apply the steps in order:

  1. Use partial fraction expansion to get \(G[z] = \frac{X[z]}{z} = \frac{c_1}{(z-2)} + \frac{c_2}{(z-3)} + \frac{c_3}{z}\).
  2. Calculate the constants \(c_1, c_2, c_3\) and get \(G[z] = \frac{(-19/6)}{z} + \frac{(3/2)}{(z-2)} + \frac{(5/3)}{(z-3)}\).
  3. Multiply by \(z\) to get \(X[z] = \frac{-19}{6} + \frac{(3/2)z}{(z-2)} + \frac{(5/3)z}{(z-3)}\).
  4. Read each piece off of the table to get the inverse z-transform of each piece to get \(x[k]\)
\[\frac{-19}{6} \rightarrow \frac{-19}{6} \delta[k]\]\[\frac{(3/2)z}{(z-2)} \rightarrow \frac{3}{2} (2)^k u[k]\]\[\frac{(5/3)z}{(z-3)} \rightarrow \frac{5}{3} (3)^k u[k]\]

This gives us that \(x[k] = \frac{-19}{6} \delta[k] + \left(\frac{3}{2} (2)^k + \frac{5}{3} (3)^k \right) u[k]\)
