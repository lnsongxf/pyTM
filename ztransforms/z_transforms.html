In this module we will introduce the idea of a z-Transform. For those of you who are familiar with Laplace transforms, the z-Transform can basically be seen as the discrete time equivalent of the Laplace transform. The z-Transform will share many of the same properties as the Laplace transform. For those of you who are not familiar with the Laplace transform then the information provided should be sufficient for your understanding.

The Z-Transform¶

The z-transform is a transformation that moves us from the time domain to the frequency domain. In the same fashion that it is possible to have both one and two-sided integral transforms the z-transform can also be one or two-sided. However, for our purposes we will just consider the one-sided z-transform.

z-Transform (one sided)
The z-transform of a sampled sequence \(x(kT)\) or \(x(k)\), where \(k\) represents non-negative integers and \(T\) is the sampling period, is defined by the following equation. Note that the complex variable \(z\) must be selected so that the summation converges [4] :
(1)\[X(z) = \mathcal{Z}[x^*(t)] = \mathcal{Z}[x(kT)] = \mathcal{Z}[x(k)] = \sum_{k=0}^\infty x(kT)z^{-k} = \sum_{k=0}^\infty x(k) z^{-k}\]

Lets practice calculating the z-transform before we discuss how it can be used.

First Example¶

Consider the unit step function defined by:

\[\begin{split}u(k) = \begin{cases} 1 & \text{if } k>0 \\ 0 & \text{else} \end{cases}\end{split}\]

We calculate the z-transform by using the definition.

\[\begin{split}X(z) = \mathcal{Z}[x(k)] &= \sum_{k=0}^\infty 1 z^{-k}\end{split}\]\[\begin{split}&= 1 + z^{-1} + z^{-2} + ...\end{split}\]\[\begin{split}&= \frac{1}{1 - z^{-1}}\end{split}\]\[\begin{split}&= \frac{z}{z-1}\end{split}\]

This z-transform is defined on \(|z|>1\).

Second Example¶

Consider the polynomial function defined by:

\[\begin{split}u(k) = \begin{cases} a^k & \text{if } k \geq 0 \\ 0 & \text{if } k < 0 \end{cases}\end{split}\]

We calculate the z-transform by using the definition.

\[\begin{split}X(z) = \mathcal{Z}[x(k)] &= \sum_{k=0}^\infty a^k z^{-k} \\ &= \sum_{k=0}^\infty (a^{-1} z)^{-k}\\ &= \frac{1}{1 - (a^{-1}z)^{-1}} \\ &= \frac{z}{z-a}\end{split}\]

This z-transform is defined on \(|z|>a\).

On Your Own¶

Now try to calculate the z-transform of the function:

\[\begin{split}u(k) = \begin{cases} \sin( \omega k T) & \text{if } k \geq 0 \\ 0 & \text{if } k < 0 \end{cases}\end{split}\]

For what values of \(z\) is it defined?

Hint

Try writing it in terms of the definition of sin. \(sin(x) = \frac{e^{ix} - e^{-ix}}{2i}\).

The Inverse Z-Transform¶

Given a z-transform \(X(z)\), it might be useful to be able to convert from the z-transform back to the time sequence that corresponds with the z-transform. This is where the inverse z-transform comes in; it moves you from the frequency domain back to the time domain.

inverse z-transform
The inverse z-transform is defined as: \(x(k) = \mathcal{Z}^{-1}[X(z)] = \frac{1}{2 \pi i} \oint_C X(z) z^{k-1} \partial z\) where the countour integration can be evaluated using the Caucy Residue Theorem. The integration contour should enclose all singularities of \(X(z)\).

Considering the complicated form of the inverse z-transform, one might hope for easier ways to calculate the inverse z-transform than using the definition. Luckily, there exist other methods of calculatig the inverse z-transform. We will briefly examine a technique where we decompose the function into a combination of functions found on the table and then can look up the corresponding inverse z-transform functions.

Steps:

  1. Break up the function through partial fraction expansion of \(G[z] = \frac{F[z]}{z}\) (This will help us get a constant term that we need).
  2. Multiply by \(z\) to get back to \(F[z]\).
  3. Put each piece into a recognizable form.
  4. Transform each piece.

We will do a quick example of how to solve by inspection using the table above.

First Example¶

Find the inverse z-transform of the function \(X[z] = \frac{8z - 19}{(z-2)(z-3)z}\).

First we use partial fraction expansion to get \(G[z] = \frac{X[z]}{z} = \frac{c_1}{(z-2)} + \frac{c_2}{(z-3)} + \frac{c_3}{z}\).

Then we calculate the constants \(c_1, c_2, c_3\) and get \(G[z] = \frac{(-19/6)}{z} + \frac{(3/2)}{(z-2)} + \frac{(5/3)}{(z-3)}\).

We then multiply by \(z\) to get \(X[z] = \frac{-19}{6} + \frac{(3/2)z}{(z-2)} + \frac{(5/3)z}{(z-3)}\).

We can read each piece off of the table to get the inverse z-transform of each piece to get \(x[k]\)

\[\frac{-19}{6} \rightarrow \frac{-19}{6} \delta[k]\]\[\frac{(3/2)z}{(z-2)} \rightarrow \frac{3}{2} (2)^k u[k]\]\[\frac{(5/3)z}{(z-3)} \rightarrow \frac{5}{3} (3)^k u[k]\]

This gives us that \(x[k] = \frac{-19}{6} \delta[k] + (\frac{3}{2} (2)^k + \frac{5}{3} (3)^k) u[k]\)

Now that the basics of a z-transform have been presented we can move onto how to use them.

Properties of the Z-Transform¶

The Z-transform has many useful properties. We will list some of theme (without proof) here. [2]

Linearity \(a_1 x_1[n] + z_2 x_2[n]\) \(a_1 X_1(z) + a_2 X_2(z)\)
Time Expansion \(x_{(k)}[n] = \begin{cases} x[r], n=rk \\ 0, n \ne rk\end{cases}\) \(X(z^k)\)
Time Shifting (backwards) [3] \(x[n-k]\) \(z^{-k}X(z)\)
Time Shifting (forwards) \(x[n+k]\) \(z^{k}X(z) - z^{k}x(0) - z^{k-1}x(1) - ... - z x(d-1)\)

Time Reversal \(x[-n]\) \(X(z^{-1})\)
Complex Conjugation \(x^*[n]\) \(X^*(z^*)\)
Differentiation \(n x[n]\) \(-z \frac{\partial X(z)}{\partial z}\)
Convolution \(x_1[n] \ast x_2[n]\) \(X_1(z) X_2(z)\)
Cross-Correlation \(r_{x_1, x_2} = x_1^*[-n] \ast x_2[n]\) \(R_{x_1, x_2}(z) = X_1^*(\frac{1}{z^*}) X_2(z)\)
Multiplication \(x_1[n]x_2[n]\) \(\frac{1}{i 2 \pi} \oint X_1(v) X_2(\frac{z}{v})v^{-1} dv\)

Present Value Calculations¶

One natural application of Z-transforms to economics is in computing the present value of an infinite income stream. Let this stream be represnted by a sequence of real numbers \(\{x_n\}_{n=0}^{\infty}\) and let \(\beta \in (0, 1)\) be a constant discount factor. The present value of this sequence is defined as

(2)\[PV(\beta) = \sum_{n=0}^{\infty} \beta^n x_n\]

Notice how Equation (2) looks very much like the definition of the z-transform (1) of \(x_n\) in terms of \(\beta\), instead of \(z\).
