Properties of the Z-Transform¶

The Z-transform has many useful properties. We will list some of theme (without proof) here. [1]

Property Definition
Linearity \(a_1 x_1[n] + z_2 x_2[n] \rightarrow a_1 X_1(z) + a_2 X_2(z)\)
Time Expansion \(x_{(k)}[n] = \begin{cases} x[r], n=rk \\ 0, n \ne rk\end{cases} \rightarrow X(z^k)\)
Time Shifting [2] \(x[n-k] \rightarrow z^{-k}X(z)\)
Scaling in z-domain \(a^n x[n] \rightarrow X(a^{-1}z)\)
Time Reversal \(x[-n] \rightarrow X(z^{-1})\)
Complex Conjugation \(x^*[n] \rightarrow X^*(z^*)\)
Differentiation \(n x[n] \rightarrow -z2 \frac{\partial X(z)}{\partial z}\)
Convolution \(x_1[n] \ast x_2[n] \rightarrow X_1(z) X_2(z)\)
Cross-Correlation \(r_{x_1, x_2} = x_1^*[-n] \ast x_2[n] \rightarrow R_{x_1, x_2}(z) = X_1^*(\frac{1}{z^*}) X_2(z)\)
Multiplication \(x_1[n]x_2[n] \rightarrow \frac{1}{i 2 \pi} \oint X_1(v) X_2(\frac{z}{v})v^{-1} dv\)

Linearity¶

\[\begin{split}a_1 x_1[n] + z_2 x_2[n] &\rightarrow a_1 X_1(z) + a_2 X_2(z)\end{split}\]

Time Expansion¶

\[\begin{split}x_{(k)}[n] = \begin{cases} x[r], & n=rk \\ 0, & n \ne rk\end{cases} &\rightarrow X(z^k)\end{split}\]

Time Shifting [2]¶

\[\begin{split}x[n-k] &\rightarrow z^{-k}X(z)\end{split}\]

Scaling in z-domain¶

\[\begin{split}a^n x[n] &\rightarrow X(a^{-1}z)\end{split}\]

Time Reversal¶

\[\begin{split}x[-n] &\rightarrow X(z^{-1})\end{split}\]

Complex Conjugation¶

\[\begin{split}x^*[n] &\rightarrow X^*(z^*)\end{split}\]

Differentiation¶

\[\begin{split}n x[n] &\rightarrow -z2 \frac{\partial X(z)}{\partial z}\end{split}\]

Convolution¶

\[\begin{split}x_1[n] \ast x_2[n] &\rightarrow X_1(z) X_2(z)\end{split}\]

Cross-Correlation¶

\[\begin{split}r_{x_1, x_2} = x_1^*[-n] \ast x_2[n] &\rightarrow R_{x_1, x_2}(z) = X_1^*(\frac{1}{z^*}) X_2(z)\end{split}\]

Multiplication¶

\[\begin{split}x_1[n]x_2[n] &\rightarrow \frac{1}{i 2 \pi} \oint X_1(v) X_2(\frac{z}{v})v^{-1} dv\end{split}\]

Theorems Associated with the Z-Transform¶

Parseval’s Theorem¶

\[\sum_{b=-\infty}^{\infty} x_1[n]x_2^*[n] = \frac{1}{i 2 \pi} \oint X_1(v) X_2^*(\frac{1}{v})v^{-1} dv\]

Initial Value Theorem¶

\[x[0] = \lim_{z \rightarrow \infty} X(z)\]

Final Value Theorem¶

\[x[\infty] = \lim_{z \rightarrow 1} (z-1) X(z)\]
[1]Many of these properties were found on this Wikipedia Page. There you can find proofs for most of them.
[2](1, 2) This property is very important in many economic contexts, as will be seen later in this module.
