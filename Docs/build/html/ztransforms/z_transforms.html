Introduction to Z-Transforms¶

In this module we will introduce the idea of a z-Transform. For those of you who are familiar with Laplace transforms, the z-Transform can basically be seen as the discrete time equivalent of the Laplace transform. The z-Transform will share many of the same properties as the Laplace transform. For those of you who are not familiar with the Laplace transform then the information provided should be sufficient for your understanding.

The Z-Transform¶

In the same fashion that it is possible to have both one and two-sided integral transforms the z-transform can also be one or two-sided. However, for our purposes we will just consider the one-sided z-transform.

z-Transform (one sided)
The z-transform of a sampled sequence \(x(kT)\) or \(x(k)\), where \(k\) represents non-negative integers and \(T\) is hte sampling period, is defined by: \(X(z) = Z[x^*(t)] = Z[x(kT)] = Z[x(k)] = \sum_{k=0}^\infty x(kT)z^{-k} = \sum_{k=0}^\infty x(k) z^{-k}\), where the complex variable \(z\) must be selected so that the summation converges.

Lets practice calculating the z-transform before we discuss how it can be used.

First Example¶

Consider the unit step function defined by:

\[\begin{split}u(k) = \begin{cases} 1 & \text{if } k>0 \\ 0 & \text{else} \end{cases}\end{split}\]

We calculate the z-transform by using the definition.

\[X(z) = Z[x(k)] = \sum_{k=0}^\infty 1 z^{-k}\]\[\sum_{k=0}^\infty 1 z^{-k} = 1 + z^{-1} + z^{-2} + ... = \frac{1}{1 - z^{-1}}\]\[\frac{1}{1 - z^{-1}} = \frac{z}{z-1}\]

This z-transform is defined on \(|z|>1\).

Second Example¶

Consider the polynomial function defined by:

\[\begin{split}u(k) = \begin{cases} a^k & \text{if } k \geq 0 \\ 0 & \text{if } k < 0 \end{cases}\end{split}\]

We calculate the z-transform by using the definition.

\[\begin{split}X(z) = Z[x(k)] &= \sum_{k=0}^\infty a^k z^{-k} \\ &= \sum_{k=0}^\infty (a^{-1} z)^{-k}\\ &= \frac{1}{1 - (a^{-1}z)^{-1}} \\ &= \frac{1}{1 - (a^{-1}z)^{-1}} \\ &= \frac{z}{z-a}\end{split}\]

This z-transform is defined on \(|z|>a\).

On Your Own¶

Now try to calculate the z-transform of the function:

\[\begin{split}u(k) = \begin{cases} \sin( \omega k T) & \text{if } k \geq 0 \\ 0 & \text{if } k < 0 \end{cases}\end{split}\]

For what values of \(z\) is it defined?

Hint

Try writing it in terms of the definition of sin. \(sin(x) = \frac{e^{ix} - e^{-ix}}{2i}\).

Simple Example of a Difference Equation¶

As a simple example of a difference equation we will examine the backward rectangle approximation of an integral. Let \(x(k)\) be the value of the integral at point \(k\) and similarly \(x(k-1)\) is the value at point \(k-1\) etc... Let the value of the function at point \(k\) be given as \(y(k)\). Then the value of the backward rectangle approximation at point \(k\) can be expressed as:

\[x_(k) = x(k-1) + T y(k-1)\]

This problem could be solved through backward substitution and obtaining the equation \(x(k) = x(0) + T \sum_{j=0}^{k-1} y(j)\), but this method will not solve all linear difference equations. Properties of the Z-Transform¶

The Z-transform has many useful properties. We will list some of theme (without proof) here. [1]

Property Definition
Linearity \(a_1 x_1[n] + z_2 x_2[n] \rightarrow a_1 X_1(z) + a_2 X_2(z)\)
Time Expansion \(x_{(k)}[n] = \begin{cases} x[r], n=rk \\ 0, n \ne rk\end{cases} \rightarrow X(z^k)\)
Time Shifting [2] \(x[n-k] \rightarrow z^{-k}X(z)\)
Scaling in z-domain \(a^n x[n] \rightarrow X(a^{-1}z)\)
Time Reversal \(x[-n] \rightarrow X(z^{-1})\)
Complex Conjugation \(x^*[n] \rightarrow X^*(z^*)\)
Differentiation \(n x[n] \rightarrow -z2 \frac{\partial X(z)}{\partial z}\)
Convolution \(x_1[n] \ast x_2[n] \rightarrow X_1(z) X_2(z)\)
Cross-Correlation \(r_{x_1, x_2} = x_1^*[-n] \ast x_2[n] \rightarrow R_{x_1, x_2}(z) = X_1^*(\frac{1}{z^*}) X_2(z)\)
Multiplication \(x_1[n]x_2[n] \rightarrow \frac{1}{i 2 \pi} \oint X_1(v) X_2(\frac{z}{v})v^{-1} dv\)

Theorems Associated with the Z-Transform¶

Theorem Mathematical Representation
Parseval’s Theorem \(\sum_{b=-\infty}^{\infty} x_1[n]x_2^*[n] = \frac{1}{i 2 \pi} \oint X_1(v) X_2^*(\frac{1}{v})v^{-1} dv\)
Initial Value Theorem \(x[0] = \lim_{z \rightarrow \infty} X(z)\)
Final Value Theorem \(x[\infty] = \lim_{z \rightarrow 1} (z-1) X(z)\)
[1]Many of these properties were found on this Wikipedia Page. There you can find proofs for most of them.
[2]This property is very important in many economic contexts, as will be seen later in this module.
